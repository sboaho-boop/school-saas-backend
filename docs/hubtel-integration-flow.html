Prepared for: Hubtel UAT Review | Date: July 2026

1. Integration Overview

EDUPLATFORM is a multi-tenant school management SaaS platform serving schools across Ghana. We integrate with Hubtel Online Checkout and Direct Receive Money API to accept payments from parents for:

  • Student Wallet Top-Up — Parents load money into their child's school wallet
  • School Subscription — Schools pay for their EDUPLATFORM subscription plan

Each school on the platform has its own Hubtel merchant account. Payments go directly to the school's account. EDUPLATFORM does not hold or process funds — Hubtel handles all payment processing.

3. Payment Flow — Wallet Top-Up (Parent)

1

Parent: Logs into EDUPLATFORM parent portal → Selects child → Clicks "Top Up Wallet"

2

EDUPLATFORM → Hubtel: Sends POST request to create checkout session

POST https://payproxyapi.hubtel.com/items/initiate { "totalAmount": 100.00, "description": "Wallet Top-Up for John Mensah", "callbackUrl": "https://your-app.com/api/wallet/hubtel-webhook", "returnUrl": "https://your-app.com/parent/dashboard?topup=success", "merchantAccountNumber": "0556674353", "cancellationUrl": "https://your-app.com/parent/dashboard?topup=cancelled", "clientReference": "WL-aB3xK9m2-1k3f7z" }
3

Hubtel → EDUPLATFORM: Returns checkout URL

{ "responseCode": "0000", "status": "Success", "data": { "checkoutUrl": "https://pay.hubtel.com/7569a11e8b784f21baa9443b3fce31ed", "checkoutId": "7569a11e8b784f21baa9443b3fce31ed", "clientReference": "WL-aB3xK9m2-1k3f7z" } }
4

EDUPLATFORM → Parent: Redirects parent to Hubtel checkout page

5

Parent: Selects payment method (Mobile Money / Bank Card / GhQR) → Enters mobile number → Receives OTP → Confirms payment

6

Hubtel → EDUPLATFORM: Sends callback with payment result

POST https://your-app.com/api/wallet/hubtel-webhook { "ResponseCode": "0000", "Status": "Success", "Data": { "CheckoutId": "59e2fbbff4e443b98e09346881ac7e9a", "ClientReference": "WL-aB3xK9m2-1k3f7z", "Status": "Success", "Amount": 100.00, "CustomerPhoneNumber": "233242825109", "PaymentDetails": { "PaymentType": "mobilemoney", "Channel": "mtn-gh" } } }
7

EDUPLATFORM: Credits student wallet balance → Returns 200 OK to Hubtel

8

Hubtel → Parent: Redirects parent back to EDUPLATFORM success page

4. Payment Flow — Wallet Top-Up via Direct Receive Money

EDUPLATFORM also uses the Direct Receive Money API for wallet top-ups. This provides a seamless experience where the parent receives a mobile money prompt directly on their phone — no redirect needed.

1

Parent: Logs into EDUPLATFORM parent portal → Selects child → Enters amount, phone number, and selects network (MTN/Vodafone/AirtelTigo)

2

EDUPLATFORM → Hubtel: Sends POST request to Direct Receive Money endpoint

POST https://rmp.hubtel.com/merchantaccount/merchants/{CollectionAccount}/receive/mobilemoney { "CustomerName": "parent@email.com", "CustomerMsisdn": "233241234567", "Channel": "mtn-gh", "Amount": 100.00, "PrimaryCallbackUrl": "https://your-app.com/api/wallet/hubtel-webhook", "Description": "Top up wallet for John Mensah", "ClientReference": "WL-aB3xK9m2-1k3f7z" }
3

Hubtel → EDUPLATFORM: Returns pending status (0001)

{ "Message": "Transaction pending. Expect callback request for final state", "ResponseCode": "0001", "Data": { "TransactionId": "09f84e20a283942e807128e8c21d08d6", "ClientReference": "WL-aB3xK9m2-1k3f7z", "Amount": 100.00, "Charges": 2.50 } }
4

Hubtel → Parent: Parent receives a mobile money prompt on their phone to approve the payment

5

Hubtel → EDUPLATFORM: Sends callback with final payment status

POST https://your-app.com/api/wallet/hubtel-webhook { "ResponseCode": "0000", "Message": "success", "Data": { "ClientReference": "WL-aB3xK9m2-1k3f7z", "TransactionId": "09f84e20a283942e807128e8c21d08d6", "Amount": 100.00, "Charges": 2.50, "AmountAfterCharges": 100.00, "PaymentDate": "2026-07-17T00:44:57.514Z" } }
6

EDUPLATFORM: Credits student wallet balance → Returns 200 OK to Hubtel

5. Transaction Status Check

If no callback is received within 5 minutes, EDUPLATFORM performs a mandatory status check:

GET https://api-txnstatus.hubtel.com/transactions/{merchantAccount}/status?clientReference={ref}

Response:

{ "message": "Successful", "responseCode": "0000", "data": { "status": "Paid", "transactionId": "7fd01221faeb41469daec7b3561bddc5", "amount": 100.0, "charges": 2.50, "amountAfterCharges": 97.50, "paymentMethod": "mobilemoney" } }
